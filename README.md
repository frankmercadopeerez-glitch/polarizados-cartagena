# Polarizados Cartagena - Web Profesional

Sitio web completo con SEO optimizado para **Polarizados Cartagena**, especialistas en películas arquitectónicas y control solar en Cartagena de Indias, Colombia.

## 📁 Estructura del Proyecto

```
polarizados-cartagena/
├── index.html              # Página principal con hero y servicios
├── robots.txt              # Control de indexación de buscadores
├── sitemap.xml             # Mapa del sitio para indexación
├── .htaccess               # Configuración del servidor para SEO y rendimiento
├── README.md              # Este archivo
│
├── contacto/
│   └── index.html         # Página de contacto y cotizaciones
│
├── servicios/
│   ├── index.html         # Página general de servicios
│   │
│   ├── polarizado-nanoceramica/
│   │   └── index.html     # Control solar con nanocerámica
│   │
│   ├── polarizado-seguridad/
│   │   └── index.html     # Películas de seguridad anti-robo
│   │
│   ├── polarizado-esmerilados/
│   │   └── index.html     # Esmerilados y privacidad
│   │
│   └── polarizado-decorativo/
│       └── index.html     # Películas decorativas personalizadas
│
└── images/                # Carpeta para almacenar imágenes
    └── (imágenes del proyecto)
```

## 🎯 Optimizaciones SEO Local para Cartagena

### Meta Tags y Geo-Targeting

- **geo.region**: CO-BOL (Colombia, Bolívar)
- **geo.placename**: Cartagena de Indias
- **geo.position**: 10.3910, -75.5138
- Palabras clave enfocadas en: "Cartagena", "Bocagrande", "Castillogrande", "Manga"

### Estructura de Datos (JSON-LD)

Incluye `@type: LocalBusiness` con:

- Nombre, teléfono, email
- Dirección completa en Cartagena
- Coordenadas geográficas
- Áreas de servicio (barrios específicos)
- horarios de atención

### URLs Amigables

```
/servicios/polarizado-nanoceramica/
/servicios/polarizado-seguridad/
/servicios/polarizado-esmerilados/
/servicios/polarizado-decorativo/
/contacto/
```

### Breadcrumbs Estructurados

Cada página incluye JSON-LD BreadcrumbList para mejor navegación en resultados de búsqueda.

## 📄 Archivos Clave

### robots.txt

- Permite indexación de todas las páginas
- Especifica sitemap.xml
- Control de velocidad de rastreo (1 segundo)

### sitemap.xml

- URLs de todas las páginas principales
- Fechas de última modificación
- Prioridades y frecuencia de cambio

### .htaccess

- Compresión GZIP para mejor rendimiento
- Caché del navegador para archivos estáticos
- Headers de seguridad
- Reescritura de URLs

## 🚀 Mejoras de Rendimiento

### Optimizaciones Implementadas

1. **CSS minimizado** con Tailwind CDN
2. **Lazy loading** de imágenes (nativas del navegador)
3. **Fuentes optimizadas** con preconnect y display=swap
4. **Caché** configurado en .htaccess
5. **GZIP** habilitado para compresión

### Recomendaciones Adicionales

- Usar imágenes WebP optimizadas en `/images/`
- Implementar CDN para archivos estáticos
- Minificar JavaScript personalizado
- Usar preload para recursos críticos
