# Carpeta de Imágenes

Esta carpeta contiene todas las imágenes del proyecto:

- Imágenes de servicios
- Fotografías de instalaciones
- Iconos personalizados
- Logos
- Fotos de testimonios

## Formatos Recomendados

- **JPG/JPEG**: Fotografías de alta calidad (comprimir con calidad 75-85%)
- **PNG**: Imágenes con transparencia, iconos
- **WebP**: Formato moderno para web (requerirá fallback JPG)
- **SVG**: Iconos y logos vectoriales

## Optimización

Todas las imágenes deben ser:

- Optimizadas para web (máximo 200KB por imagen)
- Nombradas descriptivamente (ej: `apartamento-bocagrande.jpg`)
- Etiquetadas con alt text en HTML

## Estructura Sugerida

```
images/
├── servicios/
│   ├── control-solar.jpg
│   ├── seguridad.jpg
│   ├── esmerilados.jpg
│   └── decorativo.jpg
├── proyectos/
│   ├── bocagrande-1.jpg
│   └── manga-oficina.jpg
└── testimonios/
    └── cliente-1.jpg
```
